About
-
-You start with 10 HP

-Collect blue circles for points

-Avoid red circles, if you collect them, then you lose -2 HP

-Collect green circles to regain +1 HP

Commands
-
Move player -> W,S,A,D or arrow keys.

Pause -> P

Resume -> R

Restart -> ENTER
